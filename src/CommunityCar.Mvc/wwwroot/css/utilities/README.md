# CSS Utilities

This folder contains utility CSS files for the CommunityCar application.

## Files

- `accessibility.css` - Accessibility utilities
- `animations.css` - Animation utilities
- `helpers.css` - Helper classes
- `overflow-fix.css` - Overflow fixes
- `print.css` - Print styles
- `responsive.css` - Responsive utilities
- `rtl.css` - Right-to-left language support
- `theme-utilities.css` - Theme-related utilities
